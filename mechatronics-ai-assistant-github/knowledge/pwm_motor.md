# PWM 电机控制实验

## 一、实验定位

PWM 电机控制实验是把数字输出能力扩展到执行器控制的重要实验。  
它让学生理解“占空比变化如何影响实际机械行为”。

## 二、实验目标

- 理解 PWM 波形和占空比概念
- 理解电机驱动与单片机直接驱动的区别
- 学会通过 PWM 控制电机速度
- 学会分析抖动、不转、转速异常等问题

## 三、实验器材

- 单片机开发板
- 直流电机或小风扇
- 电机驱动模块
- 外部供电
- 下载器与连接线

## 四、实验原理

PWM 通过固定周期内改变高电平持续时间占整个周期的比例，也就是占空比，来改变输出平均能量。  
占空比越大，电机等效平均电压通常越高，转速也会提高。

## 五、硬件关键点

- 电机通常不能直接由 GPIO 驱动
- 必须使用驱动模块、三极管或 MOS 管
- 电机供电往往独立于单片机供电
- 驱动电源与控制系统地线要共地

## 六、程序设计思路

1. 初始化 PWM 输出引脚
2. 配置定时器输出 PWM
3. 设置频率
4. 设置初始占空比
5. 动态调整占空比观察转速变化

### 常见控制逻辑

- 固定占空比测试
- 按键调速
- 串口输入控制转速

## 七、平台差异

### 1. 51 单片机

- 常借助定时器软件模拟 PWM
- 实现简单但精度有限

### 2. STM32

- 可直接利用定时器硬件 PWM 输出
- 适合工程控制

### 3. Arduino

- 常用 `analogWrite()` 快速实现 PWM 控制

## 八、常见问题与排查

### 1. 电机不转

- 驱动模块未使能
- 外部供电不足
- PWM 没输出
- 电机接线错误

### 2. 电机抖动

- 占空比太低
- PWM 频率不合适
- 供电不稳

### 3. 转速变化不明显

- 占空比调整范围太小
- 驱动能力不足
- 负载过大

## 九、排查建议

1. 先查驱动模块和供电
2. 再查 PWM 是否输出
3. 最后查占空比和频率设置

## 十、安全注意事项

- 电机不要直接接单片机 IO
- 外部供电极性不能接反
- 电机启动瞬间电流较大，要注意驱动能力

## 十一、报告写作建议

重点可写：

- PWM 原理

### 可直接参考的现象描述

“当 PWM 占空比逐步增大时，电机转速明显提升；当占空比较低时，电机可能无法稳定起转，说明驱动输出平均能量不足。”
